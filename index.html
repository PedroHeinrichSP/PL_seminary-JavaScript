Introdução

JavaScript é uma linguagem de programação que foi criada em 1995 por Brendan Eich. A linguagem foi originalmente concebida para tornar as páginas web mais interativas, permitindo que os desenvolvedores criassem efeitos visuais e interatividade em tempo real para o usuário final.
