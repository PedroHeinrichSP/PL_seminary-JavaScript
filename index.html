Introdução

JavaScript é uma linguagem de programação que foi criada em 1995 por Brendan Eich. A linguagem foi originalmente concebida para tornar as páginas web mais interativas, permitindo que os desenvolvedores criassem efeitos visuais e interatividade em tempo real para o usuário final. [1]

Introdução

É uma linguagem de alto nível, tornando-a fácil de aprender e implementar. Ao longo dos anos, se tornou uma das linguagens de programação mais populares da atualidade. Sendo amplamente utilizada em áreas como desenvolvimento web, aplicativos desktop, jogos e programação de microcontroladores. [2]

Paradigmas[4]



Orientada a objeto
organiza dados e funcionalidades em objetos interativos.
Imperativa
se concentra em descrever passo a passo as instruções e alterações de estado para alcançar um resultado desejado.
Declarativa
descreve o que deve ser alcançado, sem se preocupar com o passo a passo ou as instruções específicas para alcançar o resultado.

Características


Tipagem dinâmica
permite que as variáveis possam ter seu tipo alterado durante a execução do programa, sem a necessidade de declaração prévia de tipos específicos.

Suporte para programação assíncrona
permite a execução de tarefas simultâneas e não bloqueantes, por meio de callbacks, promessas, async/await e outras técnicas.

Linguagem de programação interpretada
permite que o código seja executado diretamente sem a necessidade de compilação prévia.

Considerações Finais

Em conclusão, o seminário sobre JavaScript nos proporcionou valiosos conhecimentos sobre o uso dessa linguagem. Embora tenhamos apreciado a aprendizagem, reconhecemos que sua especificidade limita sua aplicação em outros propósitos, o que pode representar um desafio. No entanto, estamos gratos por ter adquirido um entendimento mais profundo dessa ferramenta essencial no desenvolvimento web.
